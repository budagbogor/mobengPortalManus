## ğŸ‘‘ Quick Start untuk Rekruter

Sebagai rekruter, Anda memiliki akses ke dashboard canggih untuk mengelola seluruh proses rekrutmen.

### **Langkah 1: Login ke Dashboard**

1.  Buka aplikasi dan klik **"Akses Recruiter Login"**.
2.  Masukkan email dan password Anda.

### **Langkah 2: Lihat Hasil Kandidat**

-   Di dashboard utama, Anda akan melihat daftar semua kandidat yang telah menyelesaikan tes.
-   Klik pada nama kandidat untuk melihat **Score Card** detail, termasuk:
    -   Skor keseluruhan
    -   Analisis kepribadian Big Five
    -   Transkrip lengkap simulasi chat
    -   Jawaban tes logika

### **Langkah 3: Kirim Hasil via WhatsApp**

-   Di halaman Score Card, klik tombol **"Kirim Hasil via WhatsApp"** untuk mengirimkan ringkasan hasil langsung ke manajer atau tim terkait.

### **Langkah 4: Analisis & Pelaporan**

-   Akses menu **"Analytics"** untuk melihat:
    -   Grafik tren jumlah kandidat
    -   Distribusi skor
    -   Statistik performa lainnya

## ğŸŒŸ Fitur Utama untuk Rekruter

### **1. Admin Dashboard**

Kelola semua aspek rekrutmen dari satu tempat:

-   **Manajemen Bank Soal:** Tambah, edit, atau hapus pertanyaan untuk simulasi dan tes logika.
-   **Konfigurasi Role:** Sesuaikan deskripsi dan kualifikasi untuk setiap posisi yang dibuka.
-   **Pengaturan Sistem:** Atur mode tes (apakah kandidat bisa melihat skor) dan konfigurasi lainnya.

### **2. API Key Management**

-   Klik **"Pengaturan API"** untuk memasukkan atau mengganti API Key Gemini Anda.
-   Jika tidak ada API Key, aplikasi akan otomatis menggunakan **OpenRouter** sebagai fallback, sehingga aplikasi **selalu berfungsi**.

### **3. Unggah Dokumen**

-   Kandidat dapat mengunggah CV dan portofolio mereka, yang akan langsung terlampir di profil mereka di dashboard Anda.

### **4. Notifikasi Multi-channel**

-   Selain WhatsApp, sistem dapat dikonfigurasi untuk mengirim notifikasi melalui **Email** dan **SMS**.
